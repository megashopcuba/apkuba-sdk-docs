import { Tabs, TabItem } from "@astrojs/starlight/components";

## Repositorio

Agrega JitPack como repositorio en tu archivo de configuración de Gradle.

<Tabs>
  <TabItem label="settings.gradle.kts (Kotlin DSL)">
    ```kotlin
    dependencyResolutionManagement {
        repositories {
            maven { url = uri("https://jitpack.io") }
        }
    }
    ```
  </TabItem>
  <TabItem label="settings.gradle (Groovy)">
    ```groovy
    dependencyResolutionManagement {
        repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
        repositories {
            mavenCentral()
            maven { url 'https://jitpack.io' }
        }
    }
    ```
  </TabItem>
</Tabs>

## Dependencia

Agrega la dependencia en el archivo `build.gradle.kts` (o `build.gradle`) de tu módulo **app**.

<Tabs>
  <TabItem label="build.gradle.kts (Kotlin DSL)">
    ```kotlin
    dependencies {
        implementation("com.github.megashopcuba:apkuba-sdk:LATEST_VERSION")
    }
    ```
  </TabItem>
  <TabItem label="build.gradle (Groovy)">
    ```groovy
    dependencies {
        implementation 'com.github.megashopcuba:apkuba-sdk:LATEST_VERSION'
    }
    ```
  </TabItem>
</Tabs>

Reemplaza `LATEST_VERSION` con la versión indicada en el badge de JitPack:

[![](https://jitpack.io/v/megashopcuba/apkuba-sdk.svg)](https://jitpack.io/#megashopcuba/apkuba-sdk)

## Requisitos

- Android API 21+ (Android 5.0 Lollipop)
- Kotlin
- Apklis o ApKuba instalado en el dispositivo
