## Repositorio

Agrega el repositorio de MegaShopCuba en tu archivo de configuración de Gradle.

<Tabs>
  <TabItem label="settings.gradle.kts (Kotlin DSL)">
    ```kotlin
    dependencyResolutionManagement {
        repositories {
            maven { url = uri("https://repo.megashopcuba.com/releases") }
        }
    }
    ```
  </TabItem>
  <TabItem label="settings.gradle (Groovy)">
    ```groovy
    dependencyResolutionManagement {
        repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
        repositories {
            mavenCentral()
            maven { url 'https://repo.megashopcuba.com/releases' }
        }
    }
    ```
  </TabItem>
</Tabs>

## Dependencia

Agrega la dependencia en el archivo `build.gradle.kts` (o `build.gradle`) de tu módulo **app**.

<GradleInstall
  repo="https://repo.megashopcuba.com/releases"
  groupId="cu.apkuba.sdk"
  artifactId="apkuba-sdk"
/>
 
## Requisitos

- Android API 21+ (Android 5.0 Lollipop)
- Kotlin
- Apklis o ApKuba instalado en el dispositivo
