## `ApklisResponse`

Representa la respuesta de una verificación de compra, usada tanto por `ApKuba.isPurchased()` como por `Apklis.isPurchased()`.

```kotlin
data class ApklisResponse(
    var userName: String? = null,
    var status: PaymentStatus = PaymentStatus.NOT_PAYED,
    var message: Exception? = null
)
```

### Propiedades

| Propiedad | Tipo | Descripción |
|---|---|---|
| `userName` | `String?` | Nombre de usuario que realizó la compra (si está disponible) |
| `status` | `PaymentStatus` | Estado de la compra |
| `message` | `Exception?` | Mensaje de error si ocurrió una excepción |

### Métodos

#### `isValid()`

```kotlin
fun isValid(): Boolean
```

Retorna `true` si el `status` es `PaymentStatus.PAYED`.

## `ApklisLicenseResponse`

Representa la respuesta de una verificación de licencia, usada por `ApKuba.isLicensePurchased()`.

```kotlin
data class ApklisLicenseResponse(
    var licenseUuid: String? = null,
    var uuid: String? = null,
    var expiredIn: String? = null,
    var status: LicenseStatus = LicenseStatus.PENDING,
    var message: Exception? = null
)
```

### Propiedades

| Propiedad | Tipo | Descripción |
|---|---|---|
| `licenseUuid` | `String?` | UUID de la licencia |
| `uuid` | `String?` | UUID adicional del registro |
| `expiredIn` | `String?` | Fecha de expiración de la licencia |
| `status` | `LicenseStatus` | Estado de la licencia |
| `message` | `Exception?` | Mensaje de error si ocurrió una excepción |

### Métodos

#### `isValid()`

```kotlin
fun isValid(): Boolean
```

Retorna `true` si el `status` es `LicenseStatus.PAYED`.
